🔗 Connect Google Drive

One-time setup. Your data will auto-sync to Google Drive after this.

⚠️ Do this on a desktop/laptop first, then use the app on your Android phone.
Step 1 — Create Google Cloud Project

Open this link on your browser:

console.cloud.google.com

Click New Project → Name it PayTrack → Click Create

Step 2 — Enable Google Drive API

In your project, go to APIs & Services → Library

Search Google Drive API → Click it → Click Enable

Step 3 — Create OAuth Credentials

Go to APIs & Services → Credentials

Click Create Credentials → OAuth Client ID

Application type: Web application

Under Authorised JavaScript origins, add the full URL of this HTML file (e.g. file:// or your hosted URL)

Click Create — you'll get a Client ID

Step 4 — Paste Your Client ID Below

Copy the Client ID (ends with .apps.googleusercontent.com) and paste it here:
